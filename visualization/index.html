Key Formulas

SIMM_pc = √(Σ_r Σ_s ψ_rs × K_r × K_s)
Where:
r, s = risk classes within product class
ψ_rs = cross-risk-class correlation
K_r = margin for risk class r
K_b = √(Σ_k WS_k² + Σ_k Σ_l≠k ρ_kl × WS_k × WS_l)
Where:
WS_k = RW_k × s_k (weighted sensitivity)
ρ_kl = intra-bucket correlation
RW_k = risk weight
K_Delta = √(Σ_b Σ_c ρ_bc × K_b × K_c)
Where:
b, c = buckets (e.g., tenor buckets for IR)
ρ_bc = inter-bucket correlation
K_b = bucket-level margin

Portfolio Optimization

Efficient allocation reduces total margin through netting benefits

Trade Allocation Optimization Algorithm

minimize Σ_p IM_p(x)
Subject to:
Σ_p x[t,p] = 1 for all trades t (each trade in exactly one portfolio)
x[t,p] ∈ {0,1} (whole-trade assignment)

Efficient Gradient Computation (Chain Rule)

∂total_IM/∂x[t,p] = Σ_k (∂IM_p/∂S_p[k]) × S[t,k]
Where:
S_p[k] = Σ_t x[t,p] × S[t,k] - aggregated sensitivity (numpy)
∂IM_p/∂S_p[k] - from AADC kernel evaluation
S[t,k] - raw trade sensitivity (precomputed)
